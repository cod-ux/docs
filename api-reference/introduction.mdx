## Welcome

Doppler provides a simple REST API for optimizing your AI agent prompts through evals-driven improvements. The API consists of two main endpoints that allow you to submit optimization tasks and retrieve results.

## Authentication

All API endpoints require authentication using an API key passed in the request headers. You can obtain your API key by signing up at [dopplr.dev](https://dopplr.dev) and accessing it from your dashboard.

### API Key Authentication

Include your API key in the request headers:

```bash
curl -X POST "https://api.dopplr.dev/create_task" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

## Base URL

All API requests should be made to:

```
https://api.dopplr.dev
```

## Endpoints Overview

- **POST /create_task** - Submit a new prompt optimization task
- **GET /get_results/{task_id}** - Retrieve optimization results for a specific task
